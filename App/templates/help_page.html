ПРиложение для заявок на оргтехнику - основные возможности и требования

Приложение предназначено для того, чтобы работники могли оставлять заявки на покупку оргтехники руководителю.

 

Общий сценарий работы

1. Работник заполняет заявку указывая пожелания по той оргтехнике, которая ему необходима

  • Поля для заполнения
  • Заголовок вкратце описывающий заявку - текст не более 300 символов
  • Описательная часть - подробно расписанная необходимость в технике с конфигурацией до 2000 символов
  • Желаемая дата до которой критично получить оргтехнику
  • приблизительная стоимость оргтехники - в рублях от (от 0 до 1000000)

2.После заполнения система присвает заявке статус - "Необработана"

3. Администратор системы рассматривает заявку и может выставить один из двух статусов

Заявка Одобрена

Заявка отклонена

4. Пользователь может просматривать статусы своих заявок на отдельной странице

 

Доп. особенности

  • Администратор может просматривать статусы заявок всех пользователей
  • Пользователь имеет возможность отозвать(удалить) свою заявку только до момента её рассмотрения администратором
  • Администратор по итогам квартала распечатывает заявки и сдает в архив распечатанные копии
